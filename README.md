# prioect_viena
PROIECT PROGRAMARE INTERNET
Bine ai venit la **Descoperă Viena**! 
Acest site prezintă cele mai importante obiective turistice, gastronomia tradițională și cultura vibrantă a capitalei Austriei.

##Structura Proiectului
viena/
│
├── index.php              # Pagina principală
├── navbar.php             # Bara de navigare
├── footer.php             # Footer-ul site-ului
├── config.php             # Conexiunea la baza de date MySQL
├── README.md              # Documentație (acest fișier)
│
├── css/
│   └── style.css          # Stilurile CSS pentru tot site-ul
│
├── pages/
│   ├── obiective.php      # Pagina cu obiective turistice
│   ├── gastronomie.php    # Pagina cu mâncăruri tradiționale
│   ├── cultura.php        # Pagina despre cultură
│   └── contact.php        # Pagina de contact
│
└── sql/
    └── viena.sql          # Scriptul SQL pentru baza de date

## Cerințe Tehnice

Pentru a rula acest proiect, ai nevoie de:

- **PHP** versiunea 8.0 sau mai nouă
- **MySQL** sau **MariaDB**
- **Server local**: XAMPP, WAMP sau MAMP
- **Browser**: Chrome, Firefox, Edge (orice browser modern)
